Front-end boilerplate.

## Contents ##

* Bootstrap 4.x alpha
* Jade
* SASS
* Autoprefixer
* Gulp
* Gulp GA
* Favicons generator

## Getting started ##

1. Install dependencies (`npm install`)
1. Remove public/ from .gitignore

`gulp` compiles and launches webserver on localhost:8080.
